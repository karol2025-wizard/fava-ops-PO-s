========================================
GUIA RAPIDA: USAR EL .EXE EN OTRA PC
========================================

ARCHIVOS A COPIAR:
==================

1. El ejecutable:
   dist\mo_and_recipes.exe

2. Archivos de configuración (IMPORTANTE):
   .streamlit\secrets.toml
   credentials\ (toda la carpeta con los archivos JSON)

3. Archivos adicionales (si existen):
   pages\mo_and_recipes.py (por si acaso)
   shared\ (toda la carpeta)
   config.py

ESTRUCTURA EN LA PC DESTINO (MUY IMPORTANTE):
=============================================

Crea una carpeta (ej: "C:\mo_and_recipes_app") y copia:

mo_and_recipes_app\
├── mo_and_recipes.exe          <-- El ejecutable
├── pages\                      <-- CARPETA COMPLETA (OBLIGATORIO)
│   └── mo_and_recipes.py      <-- Este archivo es CRITICO
├── shared\                     <-- CARPETA COMPLETA (OBLIGATORIO)
│   ├── api_manager.py
│   ├── gdocs_manager.py
│   ├── gsheets_manager.py
│   └── [todos los otros .py]
├── config.py                   <-- OBLIGATORIO
├── .streamlit\                 <-- CARPETA (OBLIGATORIO)
│   └── secrets.toml            <-- OBLIGATORIO
└── credentials\                <-- CARPETA (OBLIGATORIO)
    ├── starship-431114-129e01fe3c06.json
    └── [otros archivos JSON si los hay]

IMPORTANTE: Todos estos archivos DEBEN estar en la misma carpeta
que el .exe. El .exe NO funcionara sin ellos.

PASOS PARA USAR:
================

1. Copia todos los archivos mencionados a la PC destino

2. Asegúrate de que el archivo .streamlit\secrets.toml tenga las rutas correctas
   a los archivos de credenciales (pueden ser rutas relativas o absolutas)

3. Ejecuta mo_and_recipes.exe haciendo doble clic

4. La aplicación se abrirá automáticamente en tu navegador en:
   http://localhost:8501

5. Si no se abre automáticamente:
   - Abre tu navegador manualmente
   - Ve a: http://localhost:8501

6. Para cerrar la aplicación:
   - Cierra la ventana de consola que se abrió
   - O presiona Ctrl+C en esa ventana

NOTAS IMPORTANTES:
==================

- El .exe puede ser grande (100-200 MB) - esto es normal
- Algunos antivirus pueden marcarlo como sospechoso (es normal con PyInstaller)
- La primera ejecución puede tardar unos segundos
- Si el puerto 8501 está ocupado, Streamlit intentará usar 8502, 8503, etc.
- Asegúrate de tener conexión a internet si la app necesita acceder a APIs externas

SOLUCION DE PROBLEMAS:
======================

Si aparece error "No module named X":
- El build puede necesitar ajustes
- Contacta al desarrollador para agregar el módulo faltante

Si no encuentra archivos:
- Verifica que todos los archivos estén en la misma carpeta
- Verifica las rutas en secrets.toml

Si no se abre el navegador:
- Abre manualmente: http://localhost:8501
- Verifica que no haya un firewall bloqueando

INSTRUCCIONES PARA GENERAR EL .EXE
===================================

OPCION 1: Usar el script Python (MAS FACIL - Recomendado)
----------------------------------------------------------
1. Abre una terminal (CMD, PowerShell, o la terminal integrada de VS Code)
2. Asegúrate de estar en el directorio del proyecto:
   cd "C:\Users\Operations - Fava\Desktop\code\fava ops PO's"
3. Ejecuta:
   python run_build.py

Este script automáticamente:
- Verificará/instalará PyInstaller
- Limpiará builds anteriores
- Generará el .exe

OPCION 2: Usar el script batch
-------------------------------
1. Abre una ventana de CMD (no PowerShell)
2. Navega al directorio del proyecto:
   cd "C:\Users\Operations - Fava\Desktop\code\fava ops PO's"
3. Ejecuta:
   build_exe.bat

OPCION 3: Usar PowerShell
-------------------------
1. Abre PowerShell
2. Navega al directorio del proyecto:
   cd "C:\Users\Operations - Fava\Desktop\code\fava ops PO's"
3. Ejecuta:
   .\build_exe.ps1

OPCION 4: Manual
----------------
1. Abre una terminal (CMD o PowerShell)
2. Navega al directorio del proyecto
3. Instala PyInstaller (si no está instalado):
   pip install pyinstaller
4. Ejecuta:
   pyinstaller mo_and_recipes.spec --clean

RESULTADO:
----------
El ejecutable estará en: dist\mo_and_recipes.exe

TIEMPO ESTIMADO: 5-10 minutos (dependiendo de tu PC)

NOTA: Si encuentras algún error durante el build, revisa el archivo
BUILD_EXE_INSTRUCTIONS.md para más detalles sobre solución de problemas.

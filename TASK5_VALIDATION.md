### PDF Features
- Professional formatting
- Actual quantity highlighted (green background, bold, larger font)
- Estimated quantity shown for reference (before actual)
- Table format with labels and values
- Footer with generation timestamp

### Summary Data Structure
```python
summary_data = {
    'mo_number': str,
    'item_code': str,
    'item_title': str,
    'lot_code': str,
    'produced_quantity': float,
    'produced_unit': str,
    'expected_output': float,      # Optional
    'expected_unit': str,          # Optional
    'date': 'YYYY-MM-DD',
    'time': 'HH:MM:SS',
    'datetime': 'YYYY-MM-DD HH:MM:SS'
}
```

### PDF Generation
- Uses ReportLab
- Letter size (8.5" x 11")
- Professional styling
- Returns BytesIO buffer for saving/printing

### Integration
- Called automatically by `ProductionWorkflow.process_production_completion()`
- PDF available in `result_data['summary_pdf']`
- Can be saved to file or sent to printer

========================================
INSTRUCCIONES RAPIDAS - MO AND RECIPES
========================================

Si tienes problemas al ejecutar la aplicación, sigue estos pasos:

1. PRIMERO: Verifica los archivos
   --------------------------------
   Ejecuta: VERIFICAR_ARCHIVOS_V2.bat
   
   (Si la ventana se cierra rápido, usa la versión V2)
   
   Esto te dirá si faltan archivos importantes.

2. SEGUNDO: Ejecuta con modo debug
   --------------------------------
   Ejecuta: EJECUTAR_CON_DEBUG_V2.bat
   
   (Si la ventana se cierra rápido, usa la versión V2)
   
   Esto mantendrá la ventana abierta para que puedas ver los errores.

3. SI SE ABREN MULTIPLES VENTANAS
   --------------------------------
   - Cierra todas las ventanas del navegador que se abrieron
   - Abre manualmente UNA sola ventana en: http://localhost:8501
   - La aplicación seguirá funcionando

4. SI HAY ERRORES
   --------------------------------
   Lee el archivo: SOLUCION_MULTIPLES_VENTANAS.md
   
   Ahí encontrarás soluciones para los errores más comunes.

========================================
ARCHIVOS IMPORTANTES
========================================

- mo_and_recipes.exe              = Ejecutable principal
- EJECUTAR_CON_DEBUG_V2.bat      = Ejecutar y ver errores (RECOMENDADO - V2)
- VERIFICAR_ARCHIVOS_V2.bat      = Verificar que todo esté bien (V2)
- SOLUCION_MULTIPLES_VENTANAS.md = Guía de solución de problemas

NOTA: Las versiones V2 mantienen la ventana abierta mejor.

========================================
EJECUTAR NORMALMENTE
========================================

Si todo está bien, puedes ejecutar directamente:
   mo_and_recipes.exe

Pero si hay problemas, usa:
   EJECUTAR_CON_DEBUG.bat
